# Uploads Directory

این پوشه برای ذخیره فایل‌های آپلود شده توسط کاربران استفاده می‌شود.

## ساختار

```
uploads/
├── avatars/        # تصاویر پروفایل کاربران
├── logos/          # لوگوهای سازمان‌ها
└── temp/           # فایل‌های موقت
```

## توجه

- این پوشه در `.gitignore` قرار دارد و فایل‌های آپلود شده در Git track نمی‌شوند
- فقط ساختار پوشه‌ها در Git نگهداری می‌شود
- در production، این پوشه باید خارج از دسترس عمومی باشد
- فایل‌ها از طریق API Route سرو می‌شوند: `/api/uploads/[...path]`

## دسترسی به فایل‌ها

فایل‌ها از طریق URL زیر قابل دسترسی هستند:

```
/api/uploads/avatars/[filename]
/api/uploads/logos/[filename]
```

## امنیت

- تمام فایل‌های آپلود شده اعتبارسنجی می‌شوند
- حداکثر حجم فایل: 2MB
- فرمت‌های مجاز: JPG, PNG, GIF, WebP
- فایل‌ها با نام یونیک (UUID) ذخیره می‌شوند
