# صفحه MeetWall - نمایش لیستی رویدادها

این صفحه برای نمایش تمام رویدادهای تایید شده به صورت لیستی طراحی شده است.

## ویژگی‌ها

### 1. **نمایش لیستی رویدادها**
- نمایش رویدادها به صورت کارت‌های زیبا
- اطلاعات کامل هر رویداد شامل:
  - تصویر
  - عنوان
  - امتیاز (ستاره)
  - موقعیت مکانی
  - قیمت
  - توضیحات مختصر
  - دسته‌بندی‌ها

### 2. **فیلترهای پیشرفته**
فیلترها در سایدبار سمت راست (در حالت RTL) قرار دارند:

- **جستجو**: جستجوی رویداد بر اساس نام
- **بازه قیمت**: انتخاب حداقل و حداکثر قیمت با اسلایدر
- **ستاره هتل**: فیلتر بر اساس امتیاز (۱ تا ۵ ستاره)
- **دسته‌بندی**: فیلتر بر اساس موضوع رویداد
- **نوع برگزاری**: حضوری، آنلاین یا ترکیبی
- **تاریخ**: انتخاب تاریخ شروع رویداد
- **تعداد تخت**: فیلتر بر اساس ظرفیت

### 3. **مرتب‌سازی**
امکان مرتب‌سازی نتایج بر اساس:
- کمترین قیمت
- بیشترین قیمت
- بالاترین امتیاز

### 4. **صفحه‌بندی**
- نمایش ۱۲ رویداد در هر صفحه
- دکمه‌های قبلی/بعدی
- نمایش شماره صفحات

### 5. **تم روشن و تاریک**
- پشتیبانی کامل از تم روشن و تاریک
- دکمه تغییر تم در هدر صفحه
- ذخیره تنظیمات تم در localStorage

### 6. **تغییر نمایش**
- دکمه تغییر به نمایش نقشه‌ای (MeetMap)
- لینک مستقیم به صفحه نقشه

## مسیر دسترسی
```
/meetwall
```

## API مورد استفاده
این صفحه از API زیر برای دریافت رویدادها استفاده می‌کند:
```
GET /api/events?status=approved&page={page}&limit=12&...filters
```

## فایل‌های مرتبط
- `page.js` - کامپوننت اصلی صفحه
- `meetwall.module.css` - استایل‌های تم روشن
- `meetwallDark.css` - استایل‌های تم تاریک

## نکات مهم
1. این صفحه فقط رویدادهای با وضعیت `approved` را نمایش می‌دهد
2. فیلترها به صورت خودکار اعمال می‌شوند
3. هنگام تغییر فیلتر، صفحه به صفحه اول بازمی‌گردد
4. تصاویر به صورت lazy load بارگذاری می‌شوند
5. در حالت خطا، دکمه "تلاش مجدد" نمایش داده می‌شود

## رسپانسیو
- **دسکتاپ**: فیلتر در سمت راست + لیست رویدادها در وسط
- **تبلت**: فیلتر بالای صفحه + لیست رویدادها پایین
- **موبایل**: نمایش به صورت عمودی با یک ستون
